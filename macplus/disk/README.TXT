UnoDOS/MacPlus - Milestone 2

This text lives in a FAT12 filesystem on the
boot floppy, read by the kernel through the
ROM .Sony driver - the same layout the x86
UnoDOS port uses, so the disk is readable on
a PC too.

Files lists the root directory. Arrow keys
move the selection; Enter opens a file in
Notepad. In Notepad, type to edit and press
Clr to save back to the floppy.
