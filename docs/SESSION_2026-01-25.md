**Build 012** - Title segment fix:

**Build 013** - Inverted text for title bar:

- Fix: Added plot_pixel_black, draw_char_inverted, gfx_draw_string_inverted

### Clock Application (Builds 013-014)

Created first real windowed application:
- apps/clock.asm - Displays RTC time (HH:MM:SS) in a window
- Uses Window Manager API to create window
- Reads BIOS RTC via INT 1Ah
- Responds to ESC key to exit

**Build 014** - INT 0x80 system call dispatch:
- Bug: Clock app crashed immediately (screen flash, return to boot)
- Cause: Kernel API functions use near `ret` but app did far `call`
- Fix: INT 0x80 now dispatches API calls (AH = function index)
  - AH=0: Return API table pointer (discovery)
  - AH=1-24: Call corresponding API function
- API table moved from 0x800 to 0x900 for code space
- Clock app simplified to use INT 0x80 (300 bytes)

## INT 0x80 Interface (v3.12.0)

Apps call kernel functions via INT 0x80:
- Set AH = function index (0-24)
- Set other registers per function requirements
- Execute INT 0x80
- Kernel dispatches to function and returns via IRET
- Results in registers per function specification

Example (clock app):
```asm
mov ah, 19              ; API_WIN_CREATE
mov bx, 100             ; X position
mov cx, 60              ; Y position
mov dx, 120             ; Width
mov si, 50              ; Height
mov al, 0x03            ; Flags
int 0x80                ; Create window
jc error                ; CF set on error
mov [win_handle], ax    ; AX = window handle
```
