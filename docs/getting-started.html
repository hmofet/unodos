Getting started

Build the image, boot it in an emulator or on real hardware, and meet the desktop.

What you need

  • A 64-bit UEFI machine (any x86-64 PC from ~2007 on), or QEMU + OVMF for an emulator.
  • To build: x86_64-w64-mingw32-gcc (UEFI apps are PE32+ images, the mingw target's native output — no gnu-efi or EDK2 needed) and python3.

Build the image

From the pc64/ directory:

./build.sh            # build the unoui desktop shell -> build/esp/
./build.sh run        # build, then boot it in QEMU + OVMF

The build compiles the shell, the toolkit, the ten themes, the apps and the drivers, links a single BOOTX64.EFI, and stages a bootable ESP tree in build/esp/ — a real FAT volume with the loader at EFI/BOOT/BOOTX64.EFI plus a few sample documents and the TrueType fonts. The default image is about 660 KB.

Run it in QEMU

./build.sh run launches QEMU with the OVMF firmware and the ESP as a virtual FAT disk. The same invocation that verifies the image is the one you use to try it:

qemu-system-x86_64 -machine q35 -m 256 \
  -drive if=pflash,format=raw,readonly=on,file=/usr/share/OVMF/OVMF_CODE_4M.fd \
  -drive if=pflash,format=raw,file=vars.fd \
  -drive format=raw,file=fat:rw:build/esp \
  -device qemu-xhci -device usb-tablet
Keyboard-firstThe whole desktop is driven from the keyboard, so QEMU needs no special input setup. For scripted/headless capture, QEMU is natively scriptable over QMP (send-key + screendump) — the screenshots in this manual were produced exactly that way.

Boot on real hardware

  1. Write the build/esp/ tree to a FAT-formatted USB stick (the loader must be at EFI/BOOT/BOOTX64.EFI).
  2. In firmware setup, boot the stick as a UEFI device. Secure Boot should be off (the image is unsigned).
  3. UnoDOS paints its splash, then the desktop. On the X1 Carbon Gen 8 the TrackPoint, touchpad and keyboard all drive it.

First boot

A splash screen with a loading bar paints while the shell and drivers come up; a short start-up chime plays as the desktop appears.

From here, everything is a keystroke away. Ctrl+Esc opens the Start menu; arrow keys and Enter launch apps; Ctrl+W closes a window. The desktop tour covers the rest.
