UnoDOS pc64

A GUI-first operating system that boots on any 64-bit UEFI PC, straight into a themed desktop with its own window manager, web browser, networking and built-in apps. No install and no command line required. This manual covers the pc64 version.

x86-64 · UEFI ~660 KB image 10 themes Keyboard-first QEMU & real-metal verified
Get started › Tour the desktop

UnoDOS is a family of GUI-first operating systems that runs on more than 20 kinds of hardware. pc64 is the modern-PC version: it runs on essentially any x86-64 PC built since about 2007, and it has been tested in emulators and on real hardware booting from a USB stick.

Just want to try it?Download unodos-pc64.iso and write it to a spare USB stick with Rufus or balenaEtcher (or boot it in a VM) - or use the one-click USB flasher. No building required. See Getting started.

What you get

A real desktop

Window manager, taskbar, desktop icons you can arrange, and windows you can move and resize, all by keyboard or pointer. The Start button, a right-click on the desktop, or Ctrl+Esc all open the programs menu.

10 live themes

A modern Aurora look (light and dark) plus eight retro skins, switchable live from the Control Panel — with proportional TrueType text and a real UI-scale setting.

Applications

A WordPad-class rich-text Editor, a real Files manager with a two-pane mode, System, Clock, a Canvas demo, plus Paint, Music, a Tracker, three games and a 3D runner.

A web browser

Shows HTML, Markdown and CSS, runs JavaScript, and loads pages over HTTP and HTTPS.

A built-in IDE

Studio lets you write, compile and run your own apps in UnoC or Python on the machine itself - a syntax-highlighting editor, a real compiler and an AI assistant, no PC needed.

Networking

Connect over Ethernet, get an address automatically, and browse the web, including secure (HTTPS) sites.

Hardware support

Your screen, keyboard, mouse and trackpad, and USB. Sound plays through the machine's audio hardware (HD Audio or AC'97), and UnoDOS drives SATA, NVMe and eMMC/SD storage with its own drivers.
