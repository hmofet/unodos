UnoDOS pc64

A GUI-first operating system that boots on any 64-bit UEFI PC, straight into a themed desktop with its own window manager, web browser, networking and built-in apps. No install and no command line required. This manual covers the pc64 version.

x86-64 · UEFI ~660 KB image 10 themes Keyboard-first QEMU & real-metal verified
Get started › Tour the desktop

UnoDOS is a family of GUI-first operating systems that runs on more than 20 kinds of hardware. pc64 is the modern-PC version: it runs on essentially any x86-64 PC built since about 2007, and it has been tested in emulators and on a real Lenovo ThinkPad X1 Carbon laptop booting from a USB stick.

Just want to try it?Most people run UnoDOS with the one-click USB flasher: download it, write UnoDOS to a spare USB stick, and boot. No building required. See Getting started.

What you get

A real desktop

Window manager, Start menu, taskbar, desktop icons, and windows you can move and resize, all by keyboard or pointer.

10 live themes

A modern Aurora look (light and dark) plus eight retro skins, switchable live from the Control Panel.

Applications

Editor, Files, System, Clock, a Canvas demo, plus Paint, Music, a Tracker, three games and a 3D runner.

A web browser

Shows HTML, Markdown and CSS, runs JavaScript, and loads pages over HTTP and HTTPS.

Networking

Connect over Ethernet, get an address automatically, and browse the web, including secure (HTTPS) sites.

Hardware support

Your screen, keyboard, mouse and trackpad, sound, and USB, plus reading files from FAT-formatted USB sticks and disks.
