Applications

The full set of apps runs on the desktop: everyday tools, creative apps, games and a 3D runner. Launch any of them from the Start menu or a desktop icon.

The Editor: real word processing

The Editor is a WordPad-class rich-text word processor. Select text and make it bold, italic or underlined; mix four typefaces and eight sizes in one document; set each paragraph left, centred or right. The document word-wraps to the window and everything is on the menu bar, the toolbar, or a shortcut (Ctrl+B/I/U, Ctrl+S/O/N, Ctrl+A, Ctrl+X/C/V). There's find & replace, a ruler, and a status bar with the cursor position.

Documents save through the Open / Save As dialog to any writable volume: the styled UWD format keeps the formatting, or name a file .TXT to save plain text.

Files: a real file manager

Files shows every mounted volume: the built-in RAM disk, the UnoDOS disk and any FAT-formatted sticks or drives. Browse into folders (Enter opens, Backspace goes up) and use the toolbar for real operations: new folder, rename, delete (it asks twice), copy and move. The Two panes button switches to a classic two-pane commander layout: copy and move then target the other pane's folder.

Everyday apps

A Canvas demo shows how an app can draw freely inside its window.

Creative tools

The games, Music and Tracker all make sound - through the machine's HD Audio or AC'97 audio hardware on modern PCs (which have no PC speaker), with the classic PC-speaker beep as the fallback on machines that still have one. The Control Panel's Volume slider sets the level.

Games

The classic games each run in their own window; Runner3D takes the whole screen (press Esc to come back to the desktop).

3D graphicsRunner3D draws real-time 3D graphics entirely in software, so it needs no graphics card.

Apps live on the disk

The games and creative tools are not baked into the system: each one is a small .UNO file in the APPS folder of the UnoDOS disk, loaded the first time you open it. Installing UnoDOS onto a PC copies them along automatically. If an app's file is missing, its window simply says so - nothing crashes.
