Applications

The full roster runs in the desktop shell — native widget apps, creative tools, games and a 3D runner. Launch any of them from the Start menu or a desktop icon.

Native widget apps

A Canvas demo shows the app-drawn widget: the toolkit owns the window chrome, focus and dragging, while the app owns every pixel inside the content rectangle — the same mechanism the games, Paint and the browser use.

Creative tools

All audio is unified on UnoSound, a single-voice PC-speaker sequencer, so the games, Music and Tracker all emit through one mixer path.

Games

The games are a native rewrite: each draws into whatever rectangle it is handed, so the same canvas fills a window or the whole screen with no letter-boxing. Press Esc to leave a full-screen game.

About the 3Duno3d is a write-once 3D pipeline with a software rasteriser (and an Intel scaffold). Runner3D drives it directly — the same portable library used across the higher-spec ports.
