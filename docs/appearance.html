Themes & appearance

Ten live themes, a Dark-mode toggle, on-demand TrueType fonts, and real resolution choice — all from the Control Panel, all without a reboot.

The Control Panel

The Control Panel is the appearance hub: a live Theme dropdown, a Dark mode toggle, a Resolution dropdown, a system-wide Font picker, a volume slider, a UI-scale spinner, and date/time spinners that set the UEFI clock. It opens on first boot and from the Start menu.

The ten themes

Aurora is the modern default — a graphics theme with soft drop shadows, anti-aliased rounded windows, a frosted taskbar and an accent underline beneath the active title bar. The other eight are pixel-faithful retro skins. Every theme re-skins the entire desktop live.

TrueType fonts

An optional font engine registers a framebuffer text provider, so choosing a TTF re-skins all UI text — titles, labels, buttons, lists — proportionally and anti-aliased, with subpixel (LCD) smoothing. Three open faces ship on the ESP (DejaVu Sans, DejaVu Sans Mono, Ubuntu) and load on demand from the FAT volume. The built-in 8×8 bitmap font stays the default and the fallback — a TTF that fails to load reverts to it cleanly.

Resolution & scaling

The desktop framebuffer is fractionally scaled to fill the panel — nearest-neighbour, aspect preserved, centred. That is how a fixed-mode laptop eDP panel still gets real resolution choice: the display mode never changes, but the desktop is rendered at the chosen size and up- or down-scaled to fit. Pick a resolution from the Control Panel and the desktop re-lays-out immediately.
