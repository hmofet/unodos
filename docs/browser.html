Web browser

A built-in web browser that shows HTML, Markdown and basic CSS, runs JavaScript, and loads pages from local disks or over the web (HTTP and HTTPS).

Files and the address bar

The browser opens on a file list that shows the built-in RAM disk and any FAT-formatted USB sticks or disks together. Type a web address in the bar at the top, or press ↓ to move into the file list; Enter opens a file and Backspace goes back.

HTML, Markdown & CSS

The browser lays out headings, word-wrapped paragraphs, bold and italic, code, links, lists and preformatted text, for both HTML and Markdown.

JavaScript

The browser runs the JavaScript on a page and adds its output to the page, with a console panel at the bottom for console.log messages.

Over the network

Type a web address and press Enter; UnoDOS connects, looks up the site, loads the page and runs any JavaScript on it.

Secure sitesSecure (https://) pages load over an encrypted TLS connection, and UnoDOS checks the site certificate against a built-in list of common certificate authorities. See Networking.
